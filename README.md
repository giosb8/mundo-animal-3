# mundo-animal-3

Aplicativo de Adoção e Proteção Animal

## Sobre o Projeto
O mundo animal é um aplicativo criado com o objetivo de ajudar a diminuir a quantidade de animais abandonados e maltratados nas ruas. A plataforma conecta pessoas que desejam adotar, doar, ajudar ou encontrar animais desaparecidos, além de incentivar a proteção e o cuidado com os animais.

##  Objetivo
O objetivo do projeto surgiu com o intuito de diminuir a grande quantidade de animais maltratados e abandonados nas ruas.  
O aplicativo busca facilitar o processo de doação e adoção de animais, além de permitir que as pessoas façam denúncias, encontrem animais perdidos e contribuam com doações ou ajuda de diferentes formas.

##  Como vai funcionar
O aplicativo fornecerá suporte para pessoas que desejam doar, adotar ou encontrar um animal desaparecido.

Ele funcionará por meio de diferentes abas:

- Adoção / Doação de animais  
- Animais desaparecidos  
- Ajuda especializada (veterinários, adestradores e passeadores)
- Doação de materiais (camas, ração, brinquedos, etc.)
- Denúncia de maus-tratos 

## Tela Inicial
A tela inicial contará com:

- Menu com acesso a todas as abas do aplicativo  
- Perfil do usuário
- Postagem
- Área com eventos gratuitos, como: castração, vacinação e feiras de adoção.

##  Funcionalidades
- Cadastro de usuários
- Publicação de animais para adoção
- Divulgação de animais desaparecidos
- Denúncia de maus-tratos
- Doação de materiais para animais
- Divulgação de eventos de cuidado animal

##  Tecnologias Utilizadas
- FlutterFlow
- Computador 
- Internet
- Banco de dados (MySQL)
- Ferramenta de design de interface (Figma)
- Sistema de login e autenticação de usuários
- GitHub
- API REST

##  Futuras melhorias
- Sistema de chat entre adotante e doador
- Geolocalização de animais perdidos
- Parcerias com ONGs e clínicas veterinárias
- Sistema de avaliação de adotantes
